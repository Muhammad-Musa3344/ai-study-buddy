# 🚀 AI Study Buddy

AI Study Buddy is a full-stack AI-powered web application that allows users to upload PDF study notes and automatically generates smart summaries and quiz questions using Generative AI.

This project is built for learning modern full-stack development with AI integration and is suitable for internship portfolio submission.

---

## 📌 Features

- 📄 Upload PDF study materials
- 🤖 AI-powered summarization of notes
- 🧠 Automatic quiz question generation with answers
- ⚡ Fast and scalable backend using FastAPI
- 🎨 Modern React-based frontend UI
- 🔗 Integration with Google Gemini AI
- 📚 Extracts text from PDFs using PyPDF

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- Tailwind CSS (for modern UI styling)

### Backend
- FastAPI
- Python
- PyPDF
- Google Generative AI (Gemini API)
